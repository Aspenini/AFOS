AFOS - Aspen Feltner Operating System

32-bit Ring 0 OS running in WASM QEMU (v86 emulator)

Ready to start

Instructions

  • Click Start OS to boot AFOS
  • Use your keyboard to interact with the OS shell
  • Type ls or dir to list files
  • Type cd <dir> to change directories
  • Press Fullscreen for a better experience
  • Known commands: ls, dir, cd, cat, run, bas, bf
